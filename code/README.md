# Cloned Repositories

## Repo 1: chatgpt-comparison-detection (HC3)
- **URL**: https://github.com/Hello-SimpleAI/chatgpt-comparison-detection
- **Purpose**: Baseline code for the HC3 dataset. Contains scripts for detecting ChatGPT-generated text and comparing it with human text.
- **Location**: `code/chatgpt-comparison-detection/`
- **Key Files**: `train.py`, `eval.py`

## Repo 2: M4 (Multi-generator detection)
- **URL**: https://github.com/mbzuai-nlp/M4
- **Purpose**: Official repository for the M4 dataset and paper. Contains code for training detectors across multiple domains and generators.
- **Location**: `code/M4/`

## Recommended Libraries (Not Cloned)
- **faststylometry**: Python library for Burrows' Delta stylometric analysis.
- **textdescriptives**: Python library for calculating various text metrics (readability, dependency distance, etc.).
- **pystyl**: Another stylometry library.
